# UniPath - NASC College Counselling and Wellness App

A Flutter-based mobile application for NASC College students to access counselling and wellness services.

## Quick Start

- **Documentation**: See `CLAUDE.md` for full architecture, features, and development guide
- **Tech Stack**: Flutter 3.12+, Firebase (Auth, Firestore, Messaging)
- **Setup**: `flutter pub get && flutterfire configure`
- **Run**: `flutter run`

## Key Features

- 📱 Student authentication (phone/email OTP)
- 🎓 Learning style assessment quiz
- 📅 Appointment booking with counsellors
- 💬 AI-powered chatbot support
- 📊 Admin analytics dashboard
- 📋 CSV report generation
